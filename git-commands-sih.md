# SIH PROJECT - Git Setup Commands

## After creating the repository on GitHub, run these commands:

# 1. Navigate to CORSAIR folder
cd "c:\Users\Prana\OneDrive\Desktop\coding\CORSAIR"

# 2. Initialize git repository
git init

# 3. Add all files
git add .

# 4. Make initial commit
git commit -m "Initial commit: CORSAIR project for SIH"

# 5. Add GitHub repository as remote (replace with your actual repo URL)
git remote add origin https://github.com/Pranvkumar/SIH-PROJECT.git

# 6. Set main branch and push
git branch -M main
git push -u origin main
