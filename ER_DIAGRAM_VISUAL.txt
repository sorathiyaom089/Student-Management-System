STUDENT ENROLLMENT FLOW:
1. Create User account (role='student')
2. Create Student record (linked to User via user_id)
3. Enroll in Courses (create Enrollment records)
4. Attend classes (create Attendance records)
5. Receive grades (create Grades records)
6. Pay fees (create/update Fees records)

FACULTY TEACHING FLOW:
1. Create User account (role='faculty')
2. Create Faculty record (linked to User via user_id)
3. Assign to Courses (create Course_Assignment records)
4. Schedule classes (create Timetable records)
5. Mark attendance (create Attendance records)
6. Enter grades (create Grades records)

ADMIN MANAGEMENT FLOW:
1. Create User account (role='admin')
2. Manage all Students, Faculty, Courses
3. Post Announcements
4. Generate Reports from all tables
